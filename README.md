ІА-32 Бойко Денис
Завдання:
Необхідно реалізувати систему, яка складається із банківського рахунку та операцій зняття та депозиту коштів.  Банківський рахунок не може йти в мінус, тому перед тим як знімати кошти необхідно дочекатися, поки кошти на рахунку з’являться

тільки нащадки Thread

Висновок з описом:
Під час виконання лабораторної роботи було реалізовано систему банківського рахунку з використанням механізмів синхронізації. Для створення потоків використано виключно наслідування від класу Thread. Додано механізми синхронізації потоків за допомогою wait() та notifyAll(), що дозволяє уникнути зняття коштів при недостатньому балансі.
